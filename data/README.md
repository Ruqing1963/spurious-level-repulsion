# data/

`ssw_central_dates.csv` — central dates of the 41 major Northern-Hemisphere
sudden stratospheric warmings (SSWs) used in Hazard 2.

**Source:** NOAA Sudden Stratospheric Warming Compendium (Butler, Sjoberg,
Seidel & Rosenlof, 2017, *Earth Syst. Sci. Data* 9, 63–76), JRA-55 column,
1958–2023. Table prepared by A. H. Butler.
<https://csl.noaa.gov/groups/csl8/sswcompendium/majorevents.html>

These dates are public-domain U.S.-government data. They are also embedded in
`code/spurious_goe.py` (the authoritative copy the analysis reads); this CSV is
provided for transparency and convenience. All other demonstrations in the
repository are synthetic and need no input data.
